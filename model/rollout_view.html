code sensitivity 看点：base 和 fast 是配对样本——同一个关卡 seed、同一串动作，唯一区别是喂给模型的 coinrun.cpp 里 maxspeed 不同（0.5 vs 0.9）。如果模型真的读懂了 code，fast 的生成画面里玩家水平移动应该更快。对比两个生成视频右侧的玩家移动速度即可判断模型是否对 code 敏感。
